EXPLORE THE MAP
To be able to explore the map, you first need to purchase and install the Minecraft PC edition. Once you have installed Minecraft, follow the steps below:

1. Start up the Minecraft program, in Minecraft version 1.9 - 1.16.2(latest)
2. Click “Multiplayer”
3. Click “Add Server”
4. For the Server Name, you can type anything you want. We suggest MiddCraft!
5. In the Server Address field, use: play.middcraft.com


You can now join the world and explore the MiddCraft map. You will not be able to chat with other players or make changes to the map. If you are associated with Middlebury, you can register with us to gain these privileges and join the community.


Note: The server is only accessible via the Minecraft PC edition (and not 'Bedrock' or by phone).
