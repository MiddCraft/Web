How do I sign up for a Minecraft Account?

Head over to minecraft.org and create an account. After you create an account, Mojang requires a one-time fee of $27 to download the app, and then you have minecraft for the rest of your life! If funding is a concern for you, please reach out to us and we will see what we can do. Making this server accessible to everyone is a top priority for us!
How did MiddCraft create the base for its map?

MiddCraft used open source LIDAR data from the Vermont Geodata portal and the WorldPainter app to create the base for its map.

What server host does MiddCraft use?

The MiddCraft server is hosted on DigitalOcean.

How many players can MiddCraft support?

MiddCraft can support up to 100 players on the server at a time.
